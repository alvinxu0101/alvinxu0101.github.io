怎样通过hexo+markdown+github发布文章

发表于 2018-01-20 | 分类于 技术 |
字数统计: 215 | 阅读时长 ≈ 1

 第一篇个人博客的文章，相对而言会写的比较乱，主要记录遇到的以下坑

  •  更换npm镜像为国内的镜像源：
    npm install -g cnpm --registry=https://registry.npm.taobao.org
  • 新建一篇博客，在打开git bash here,输入：
    hexo new "博文名字"
  • 然后就会生成一个以.MD结尾的文件，通过MD编辑器打开进行编辑，编辑好了之后就可以通过hexo命令发布到github博客上了。
  • 命令依次如下所示：
    hexo clean
    hexo g
    hexo d
  • 让博客文章只显示一部分，点击阅读全文进入全文阅读，在md编辑文章时在你想要在首页显示的地方加上以下内容是一种方式：
    <!--more-->
  • 还有一种方式是在主题配置文件中修改：
    auto_excerpt: enable: false length: 150（可自行设置）
