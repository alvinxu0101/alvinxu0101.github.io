IDEA无法登录github解决

发表于 2018-09-02 | 分类于 IDEA |
字数统计: 144 | 阅读时长 ≈ 1

  今天新装了系统以及idea，在系统中配置了SSH，并且也在github上进行了配置。但是当使用IDEA登录github时无法登录，显示错误信息invalid authentication data。网上找了很久没有解决办法，后来通过stackoverflow上看见应该是token的问题，就上自己的github上看了一下token，发现有了很多的token，就把已有的全部删除了，然后再次通过IDEA登录github就可以登录了，记录一下，以防以后再次遇到此种情况不知道怎么解决。
